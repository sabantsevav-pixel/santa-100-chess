# ♔ Santa Chess 100 ♚

A 10×10 chess variant featuring the Musketeer piece — playable online with real-time multiplayer!

**[https://YOUR-USERNAME.github.io/grand-musketeer-chess/](https://YOUR-USERNAME.github.io/grand-musketeer-chess/)**

## 🆕 What's Different from Standard Chess?

| Feature | Standard Chess | Grand Musketeer Chess |
|---|---|---|
| Board | 8×8 | 10×10 |
| New Piece | — | Musketeer (moves 2 squares any direction) |
| Knights | L-shape (2+1) | Standard + Extended (3+1) |
| Pawns | Move 1-2 squares first move | Move 1-3 squares first move |
| Castling | King moves 2 squares | King moves 2, 3, or 4 squares |
| En Passant | Standard | Extended (captures on any passed square) |

## 🎯 How to Play

### Local (Same Device)
1. Click **"Play Local"**
2. Take turns on the same screen

### Online Multiplayer
1. **Player 1:** Click **"Create Game"** → share the 6-letter code
2. **Player 2:** Enter the code → click **"Join Game"**
3. Play in real-time!

## 🏰 The Musketeer Piece
- Moves up to **2 squares** in any direction (horizontal, vertical, diagonal)
- Cannot jump over other pieces
- Each side starts with **2 Musketeers**
- Denoted by **M** in notation

## 📁 Tech Stack
- Pure HTML/CSS/JavaScript (no frameworks)

- SVG chess pieces (cburnett set + custom Musketeer)
